Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide

AllergyIntolerance (NCD TH)

ใช้บันทึกข้อมูลการแพ้ยาแพ้อาหาร
Condition (NCD TH)

ใช้บันทึกปัญหา (problem list) หรือการวินิจฉัย (diagnosis)
Coverage (NCD TH)

ใช้บันทึกข้อมูลสิทธิ์การรักษา
DiagnosticReport Profile for Lab (NCD TH)

ใช้รายงานผลการตรวจแล็บที่มีหลายรายการย่อย (การจัดกลุ่ม Observation เช่น lab panel)
Encounter (NCD TH)

ใช้บันทึกข้อมูลการรับบริการสุขภาพ
EpisodeOfCare (NCD TH)

ใช้จัดกลุ่มของ Encounter ประเภทเดียวกัน เช่น Encounter ทั้งหมดในการดูแลโรคเบาหวาน
FamilyMemberHistory (NCD TH)

ใช้บันทึกประวัติการเจ็บป่วยในครอบครัว
Media (NCD TH)

ใช้บันทึกรูปประกอบ ในที่นี้เช่น แผนผังครอบครัว
Observation Profile - Lab (NCD TH)

ใช้บันทึกข้อมูลผลการตรวจแล็บแต่ละตัว
Observation Profile - PE & Survey (NCD TH)

ใช้บันทึกข้อมูลการตรวจร่างกาย และการประเมินตามแบบประเมินต่าง ๆ
Observation Profile - Social (NCD TH)

ใช้บันทึกข้อมูลประวัติทางสังคมของผู้ป่วย
Observation Profile - Vital Signs (NCD TH)

ใช้บันทึกข้อมูล vital signs
Organization (NCD TH)

ใช้บันทึกข้อมูลหน่วยงาน/องค์กร
Patient (NCD TH)

ใช้บันทึกข้อมูล demographic ของผู้ป่วย เช่น ชื่อ เลขประจำตัว ที่อยู่ เป็นต้น

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide

NCD TH Laboratory Value Set

รหัสมาตรฐานการตรวจทางห้องปฏิบัติการที่ใช้ใน NCD IG
NCD TH Vital Signs Value Set

รหัสจากระบบ LOINC ในการบันทึกชนิดการตรวจ vital signs
Thai Government Coverage Type Value Set

รหัสประเภทสิทธิ์การรักษาพยาบาลภาครัฐของไทย รวมกับรหัสที่ HL7 กำหนด

Terminology: Code Systems

These define new code systems used by systems conforming to this implementation guide

Thai Government Coverage Type Code System

รหัสประเภทสิทธิ์การรักษาพยาบาลภาครัฐของไทย
